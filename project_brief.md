# Project Brief — Industrial Predictive Maintenance Portfolio Project

## Goal
Build from scratch a professional portfolio project focused on industrial predictive maintenance.

The project should be realistic, technically solid and useful for strengthening my profile for roles such as:

- AI Engineer
- ML Engineer
- Applied AI Engineer
- Industrial AI Engineer

## Current starting point
This project is starting from zero.

At this stage:
- the dataset has not been selected yet
- the exact problem formulation is not closed yet
- the MVP is not defined yet

So the first part of the work should focus on:
1. selecting a good dataset
2. choosing the right problem formulation
3. defining a realistic MVP
4. building the project step by step

## What I want from this project
I want the final result to be:

- good for GitHub
- defendable in interviews
- technically credible
- clearly related to industrial AI
- structured like a real ML/AI project, not only as a notebook exercise

## Preferred development approach
The project should be built in phases:

1. dataset selection
2. problem definition
3. MVP definition
4. exploratory analysis
5. target construction
6. baseline model
7. improved model
8. repository and documentation
9. API/app/deployment layer

## Preferred stack
- Python
- pandas
- numpy
- matplotlib
- scikit-learn
- XGBoost or LightGBM
- FastAPI
- Streamlit
- pytest
- Docker

## Main requirement
Always prioritize the most practical and portfolio-friendly path.
